LN Pagos Latam

mayo 15, 2023

  • Wordpress
  • E-Commerce
  • Bitcoin Lightning

Un plugin de Wordpress que permite aceptar pagos con Bitcoin Lightning canalizados directamente a la cuenta en Buda del vendedor.

De esta forma, el vendedor puede ofrecer pagos en Bitcoin Lightning y al mismo tiempo recibir el pago en moneda local, al vender el bitcoin y transferir el monto desde el exchange hacia la cuenta bancaria del vendedor. El repositorio de Github es el siguiente:

https://github.com/felipebrunet/lnpagos-latam
